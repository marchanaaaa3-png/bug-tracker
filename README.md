# Bug Tracker Project 🐞

A simple full-stack bug tracking application built using Node.js, Express, and MongoDB.

## 🚀 Features
- Add new bugs
- View all bugs
- Set priority (Low, Medium, High)

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB
- HTML, CSS

## ⚙️ Setup Instructions

1. Clone the repo
2. Run:
   npm install
3. Add your MongoDB URI in .env
4. Start server:
   npm start
